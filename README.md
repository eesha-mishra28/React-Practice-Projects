# React Practice Projects

This repository contains three separate React + Vite practice apps:

- Bharat Clock - a clock and date display app
- Calculator - a basic calculator UI
- To-Do - a simple task management app

## Projects

### Bharat Clock
Path: `Bharat-Clock/clock`

Run it with:

```bash
cd Bharat-Clock/clock
npm install
npm run dev
```

### Calculator
Path: `Calculator/calci`

Run it with:

```bash
cd Calculator/calci
npm install
npm run dev
```

### To-Do
Path: `To-Do/my-app`

Run it with:

```bash
cd To-Do/my-app
npm install
npm run dev
```

## Tech Stack
